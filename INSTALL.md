# راهنمای نصب ماژول Call Now Button برای Joomla

## پیش‌نیازها

- Joomla 5.0 یا بالاتر (سازگار با Joomla 6)
- PHP 8.1 یا بالاتر
- مرورگرهای مدرن (Chrome, Firefox, Safari, Edge)

## روش نصب

### روش 1: نصب از طریق پنل مدیریت

1. فایل `mod_callnowbutton.zip` را دانلود کنید
2. وارد پنل مدیریت Joomla شوید
3. به **Extensions > Manage > Install** بروید
4. روی تب **Upload Package File** کلیک کنید
5. فایل ZIP را انتخاب و آپلود کنید
6. منتظر بمانید تا نصب کامل شود

### روش 2: نصب دستی

1. فایل ZIP را استخراج کنید
2. پوشه `mod_callnowbutton` را در مسیر `/modules/mod_callnowbutton` کپی کنید
3. وارد پنل مدیریت Joomla شوید
4. به **Extensions > Discover** بروید
5. روی **Discover** کلیک کنید
6. ماژول "Call Now Button" را پیدا کرده و نصب کنید

## فعال‌سازی و پیکربندی

1. به **Extensions > Modules** بروید
2. روی **New** کلیک کنید
3. ماژول "Call Now Button" را انتخاب کنید
4. تنظیمات را پیکربندی کنید:

### تنظیمات ضروری:
- **Enable Button**: Yes
- **Phone Number**: شماره تلفن خود را با کد کشور وارد کنید (مثال: +989123456789)

### تنظیمات پیشنهادی:
- **Button Position**: Bottom Right (پایین راست)
- **Button Color**: #25D366 (سبز WhatsApp)
- **Display Mode**: Mobile Only (فقط موبایل)

5. در تب **Menu Assignment** صفحاتی که می‌خواهید دکمه نمایش داده شود را انتخاب کنید
6. روی **Save & Close** کلیک کنید

## تست

1. به وب‌سایت خود بروید
2. در یک دستگاه موبایل یا با استفاده از Developer Tools مرورگر (Device Mode) صفحه را باز کنید
3. دکمه تماس فوری باید در موقعیت تعیین شده نمایش داده شود
4. روی دکمه کلیک کنید تا تماس برقرار شود

## عیب‌یابی

### دکمه نمایش داده نمی‌شود:

1. بررسی کنید که ماژول فعال است
2. بررسی کنید که شماره تلفن وارد شده است
3. بررسی کنید که در حالت Mobile Only نیستید (اگر از دسکتاپ تست می‌کنید)
4. Cache را پاک کنید (System > Clear Cache)
5. بررسی کنید که ماژول در صفحات مورد نظر Assign شده است

### دکمه در موقعیت اشتباه است:

1. تنظیمات Position را بررسی کنید
2. بررسی کنید که CSS با تم شما تداخل ندارد
3. Z-Index را افزایش دهید

### رنگ دکمه تغییر نمی‌کند:

1. Cache مرورگر را پاک کنید
2. بررسی کنید که CSS ماژول به درستی لود می‌شود
3. در Developer Tools بررسی کنید که استایل‌های inline اعمال شده‌اند
