# تغییرات ماژول Call Now Button

همه تغییرات قابل توجه این پروژه در این فایل مستند می‌شوند.

فرمت بر اساس [Keep a Changelog](https://keepachangelog.com/fa/1.0.0/) است،
و این پروژه از [Semantic Versioning](https://semver.org/lang/fa/) پیروی می‌کند.

## [1.0.0] - 2024-01-01

### اضافه شده
- انتشار اولیه ماژول Call Now Button برای Joomla
- سازگاری کامل با Joomla 5 و 6
- پشتیبانی از 8 موقعیت مختلف دکمه
- تنظیمات رنگ، اندازه و موقعیت دکمه
- پشتیبانی از 3 نوع ظاهر (دایره‌ای، عرض کامل پایین، عرض کامل بالا)
- کنترل نمایش در صفحات مختلف
- پشتیبانی از نمایش فقط موبایل، فقط دسکتاپ یا همه دستگاه‌ها
- ردیابی کلیک در Google Analytics (3 نوع: gtag, analytics, classic)
- ردیابی تبدیل در Google Ads
- پشتیبانی چندزبانه (انگلیسی و فارسی)
- طراحی ریسپانسیو و مدرن
- انیمیشن‌های نرم و حرفه‌ای
- دسترسی‌پذیری (Accessibility) کامل

### ویژگی‌های فنی
- استفاده از Namespace و PSR-4
- ساختار MVC مدرن Joomla
- Service Provider Pattern
- Helper Factory Pattern
- Module Dispatcher Pattern
- امنیت کامل (XSS Protection, Input Sanitization)
- بهینه‌سازی عملکرد
